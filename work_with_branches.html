Работа с ветками: git branch - позволяет посмотреть текущую ветку git branch -v - позволяет посмотреть текущую ветку, хэш, коммит git branch develop1 - генерация ветки с именем develop1 git checkout develop1 - переход в ветку develop1 git branch -m master - переименование текущей ветки на "master" git checkout -b develop1 - позволяет создать ветку develop1 и сразу перейти в нее git checkout hash_commita - позволяет перейти к коммиту с хэшом "hash_commita" (здесь внимательнее, переход по коммитам, вернуться на последний коммит != попасть на актуальную версию проекта)
