Для отмены изменения в коде, возврашения к коммиту, команлы для терминала и способ через гит-граф: Откат - если нажал ctrl + s до коммита, можно в source control под кнопкой commit нажать на discard all changes или через многоточее changes -> discard all changes - через консоль: git checkout -- [название файла] - для отдельного файла или git checkout -- . - для всех файлов - если файлы попали в staged changes (кэш коммита условно) и нужно откатить, то для этого нужно нажать на unstage all changes или через многоточие changes -> unstage all changes - через консоль: git reset [название файла] - для извления отдельного файла из stage или git reset . - для извлечения всех файлов из stage после чего можно сделать discard all changes Откат коммита (до его пуша) - после того, как уже закоммитил что-то, но хочешь вернуть коммит предыдущий git reset --hard HEAD^1
