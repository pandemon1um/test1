10)
Для выделения текста жирным, нужно пометить нужный фрагмент 2-мя звездочками по бокам. **Вот таким образом**. Если нужен курсив, то это по 1 звезде по бокам *Вот так вот*. Или жирный курсив через 3 звезды: ***вот так вот***. Можно также делать не через звезды, а через нижнее подчеркивание _ _вот так вот_.

11) Если использовать обратный апостроф, можно выделять текст или код по блокам
`
$абвгд; 
$еёжзи; 
$клмно;
`  
Можно добавить по три обратных апострофоф, то получится целый блок:
```
$абвгд; 
$еёжзи; 
$клмно;
```
